Agents for hub_notify

This folder contains agent definitions and prompt templates to guide Copilot-agent-style assistants for common development workflows in this repository.

Included agents:
- `code-assistant` — producing implementation code and small refactors
- `pr-review` — generating PR review comments and checklists
- `test-runner` — creating tests and fixtures
- `commit-helper` — crafting commit messages and changelog entries

Usage:
Open the appropriate agent file and copy the prompt templates when asking Copilot for focused help.
