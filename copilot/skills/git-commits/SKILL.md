Skill: Git Commit Message Templates

Description:
Standardized commit message templates to keep history clear and helpful.

Templates:
- "feat(channels): add send_push helper for push notifications"
- "fix(queue): handle empty payloads in consumer"
- "chore(deps): bump pytest to 7.x"

Prompt examples:
- "Generate a concise commit message for: added input validation to app/channels/email.py and unit tests covering invalid email addresses."
