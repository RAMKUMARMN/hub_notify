Copilot Skills for hub_notify

This folder contains concise skill definitions to help GitHub Copilot (and developers) produce better, more consistent code for this repository. Each skill includes when to use it, quick prompt templates, and examples.

Skills included:
- python-project — project conventions and setup prompts
- prompt-engineering — templates for instructing Copilot effectively
- testing-guidelines — unit/integration testing prompts and patterns
- code-review — PR and code review guidance prompts
- git-commits — commit message templates and examples
